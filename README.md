# ECommerceWeb
ECommerceWeb is a full-stack e-commerce web application built with C#, ASP.NET, and modern frontend technologies. It allows users to browse products, manage carts, place orders, and provides an admin panel for managing products and users. This project is ideal for learning full-stack development, building a portfolio, or extending with advanced features.

## Demo

Check out the live application here: [ECommerceWeb Live Demo](https://nile-store-swe.vercel.app/)

## Features

- User Authentication: Sign Up, Login, Logout
- Browse Products: By Categories & Search
- Shopping Cart: Add, Remove, Edit Items
- Order Management: Place Orders and View History
- Admin Panel: Manage Products & Users
- Responsive Design: Desktop & Mobile

## Getting Started

### Prerequisites

- Git
- .NET SDK
- Node.js & npm
- SQL Server or compatible database

### Installation

1. Clone the repository:

```bash
git clone https://github.com/muhamedashraf04/ECommerceWeb.git
cd ECommerceWeb
```
2.   Configure the database connection in `backend.env`.
    
3.  Run backend:
```bash
dotnet run
```
4. Navigate to the frontend folder and install	 dependencies:
```bash
cd frontend
npm install
npm start
```
5. Open your browser and go to `http://localhost:3000` (or your configured port).

 or simply go to that link:
  (https://nile-store-swe.vercel.app/)
