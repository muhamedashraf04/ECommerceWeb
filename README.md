## Overview

**ECommerceWeb** is a full-stack e-commerce web application built with **C#**, **ASP.NET**, and frontend technologies. It allows users to browse products, manage carts, place orders, and provides admin functionality for managing products and users.  

This project is perfect for learning full-stack development, building a portfolio, or extending with advanced features.

## Features

- User Authentication (Sign Up, Login, Logout)
- Browse Products by Categories & Search
- Shopping Cart with Add/Remove/Edit Items
- Place Orders and View Order History
- Admin Panel for Product & User Management
- Responsive Design for Desktop & Mobile
